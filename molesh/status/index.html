現在の状態

  • Android v0.1 のローカル実装とビルド確認は完了しています。
  • アプリ名は「モレッシュ」、英字表記は「Molesh」です。
  • パッケージ名は com.molesh.app、versionName は 1.0.0、versionCode は 1 です。
  • Play Console のアカウント確認完了後に、初回アプリ作成と公開準備を進める段階です。

現在の公開方針

  • 近くのトイレ候補をGoogleマップで開く小さなAndroidアプリとして公開します。
  • アプリ本体では位置情報を取得しません。
  • アプリ本体では個人情報を収集しません。
  • 初回リリース時点では、広告、課金、ログイン、Firebase、Analytics、Google Places APIを使いません。
  • 検索結果はGoogleマップ側で表示されます。

完了済み

  • アプリ名、パッケージ名、公開用メール、固定文言の整理。
  • Googleマップ検索URLを外部アプリまたはブラウザで開く実装。
  • Googleマップが開けない場合のブラウザfallbackとエラー表示。
  • 位置情報権限なし、広告なし、課金なし、外部SDK最小構成の確認。
  • debug APK と release AAB の生成確認。
  • スマートフォン用スクリーンショットの取得。
  • ストア掲載文、Play Console入力メモ、プライバシーポリシー案の作成。
